# Markdown example

## Inline formatting

We support the common inline formatting, like *emphasis*, **strong**, ~~strikethrough~~, `inline code`, and $inline math$.
We also support [links](https://hipphampel.de) and ![images](https://github.com/markdown-it/markdown-it-image-html).

## Code and quote blocks

### Code 
We understand code blocks:
```text
This is a code block.
We ignore the language. 
So no syntax highlighting, sorry.
```

### Simple quote
The following is a *simple* quote:
> To be or not to be, that is the question.
> Manus manum lavat.
> Last christmas I gave you my heart.

### Quotes with markers
We understand some markers for quotes:

#### Tip
> [!TIP]
> This is a tip.
> 
> Take it or leave it.

#### Note
> [!NOTE]
> This is a note.
>
> Do you want to know more?.

#### Important
> [!IMPORTANT]
> This is an important note.
>
> Please obey me!

#### Warning
> [!WARNING]
> This is a warning
>
> Be aware!

#### Caution
> [!CAUTION]
> This is an error
>
> Something bad happened!
